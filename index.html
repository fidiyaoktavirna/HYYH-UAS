Future

☻ Proyeksi 7 tahun ke depan: Dapat bepergian ke luar negeri, entah itu untuk belajar, bekerja, ataupun hanya untuk berlibur. Tetapi hal yang paling dekat dengan saya saat ini adalah belajar. Saya ingin menempuh pendidikan setidaknya untuk satu semester di Korea Selatan dengan menggunakan beasiswa. Untuk lingkup pekerjaan, saya ingin memiliki pekerjaan yang tidak memerlukan kekuatan fisik seperti terjun langsung di lapangan. Saya memiliki impian untuk menjadi arsiparis atau seorang data analyst, dimana pekerjaan tersebut berkesinambungan dengan program studi yang saya tempuh saat ini. Saya juga berminat untuk bekerja di bidang kreatif, jadi saya pun memiliki impian untuk menjadi copywriter atau graphic desainer di suatu perusahaan.

☻ Hal yang harus dilakukan: Untuk dapat menempuh pendidikan di luar negeri, saya bertekad untuk mencari beasiswa yang dapat mendukung saya untuk belajar di Korea Selatan, seperti misalnya program beasiswa IISMA. Selain mencari program beasiswa yang sesuai dengan kebutuhan saya, saya juga sedang mempelajari bahasa Korea melalui aplikasi belajar bahasa online gratis, duolingo. Tak hanya melalui duolingo, saya juga belajar bahasa Korea secara tidak langsung dari hobi saya yang gemar menonton variety show Korea, dan juga kegemaran saya dalam mengidolakan artis Korea. Dengan menonton konten-konten artis favorit saya, secara tidak langsung saya dapat belajar mengenai bahasa maupun budaya Korea. Sedangkan untuk dapat bekerja di bidang yang saya impikan, tentunya saya harus menggali ilmu sebanyak yang saya bisa selama di perkuliahan ini karena akan sangat membantu untuk mencapai pekerjaan tersebut. Dan juga mengikuti course maupun webinar perihal data analyst. Saya gemar untuk mendesain menggunakan software, meskipun belum begitu mahir tapi saya ingin mempelajari lebih dalam lagi mengenai dunia desain grafis.

☻ Langkah dan pencapaian tiap tahun: Mengikuti course atau webinar, mengikuti magang program kampus merdeka, mengikuti kepanitiaan atau organisasi di perkuliahan dengan bidang yang memiliki kesinambungan terhadap mimpi saya, menyelesaikan pembelajaran di duolingo, mengikuti tes TOPIK (Test of Proficiency in Korean), mendaftar beasiswa.
